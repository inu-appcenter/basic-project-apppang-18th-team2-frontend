## 1. 로그인 `/login`

### 1.1 이메일 입력
- 이메일 형식 유효성 검사 (@ 포함, 도메인 형식)
- 입력 중 우측 X 버튼 노출 → 클릭 시 입력값 초기화
- 포커스 시 필드 하이라이트

### 1.2 비밀번호 입력
- 기본값: 마스킹 처리
- 우측 아이콘 클릭 시 평문 토글
- 최소 8자 이상 validation

### 1.3 로그인 버튼
- 이메일 + 비밀번호 둘 다 입력된 경우에만 활성화
- 클릭 시 API 호출
- 성공: Access Token / Refresh Token 저장 후 메인 이동
- 실패: 에러 메시지 Toast 또는 Inline 노출

**JWT 정책**
- `accessToken`: localStorage 저장
- `refreshToken`: cookie 저장 (백엔드 설정)
- 앱 재실행 시 자동 로그인 (토큰 유효성 검사 후 메인 이동)
- "자동로그인" 체크박스 구현 X → 무조건 자동로그인

### 1.4 아이디·비밀번호 찾기
- 별도 계정 찾기 페이지로 이동
- 이메일 인증 기반 비밀번호 재설정 플로우

### 1.5 회원가입
- 회원가입 페이지로 라우팅
- Outlined 버튼 스타일 (로그인 버튼과 시각적 구분)

### 1.6 뒤로가기 버튼
- 이전 페이지로 이동
- 스택 없을 경우 메인 페이지(비로그인)로 이동

## 2. 회원가입 `/register`

### 2.1 이메일 입력
- 이메일 형식 유효성 검사
- 포커스 아웃 시 실시간 중복 확인 API 호출
- 중복 시 인라인 에러: "이미 사용 중인 이메일입니다."

### 2.2 비밀번호 입력
- 마스킹 기본, 우측 토글 아이콘
- 최소 8자 이상 + 영문+숫자 조합
- 조건 미충족 시 인라인 에러 노출

### 2.3 이름 입력
- 한글/영문만 허용 (숫자·특수문자 차단)
- 최소 2자 이상

### 2.4 휴대폰 번호 입력
- 숫자만 입력, 자동 하이픈 포맷팅: `010-0000-0000`
- SMS 인증 연동 여부 → 별도 기획 협의

### 2.5 약관 동의
- 클릭 시 하위 개별 약관 전체 체크/해제 토글
- 하위 약관 모두 체크 시 "모두 동의" 자동 활성화
- > 버튼 클릭 시 약관 상세 모달 or 별도 페이지
- 필수 항목 미동의 시 회원가입 버튼 비활성화

### 2.6 회원가입 버튼
활성화 조건 (AND):
- 이메일 유효 + 중복 없음
- 비밀번호 정책 충족
- 이름 입력 완료
- 휴대폰 번호 입력 완료
- 필수 약관 전체 동의

클릭 시:
- API 호출 + 로딩 스피너 (중복 클릭 방지)
- 성공: 회원가입 완료 페이지 or 자동 로그인 후 메인
- 실패: 에러 케이스별 메시지 처리

## 4. 상품 리스트 `/products`

### 4.1 헤더
- 좌측 뒤로가기 버튼
- 상단 중앙: 카테고리명 or 검색 키워드 (진입 경로 따라 동적 변경)

### 4.2 정렬 및 뷰 전환
- 정렬 드롭다운: 랭킹순 / 최신순 / 최저가순 / 최고가순
- 뷰 전환 버튼 구현 X (리스트 형태만)

### 4.4 상품 카드 리스트
리스트 뷰 구성:
- 좌측 썸네일 + 우측 상품 정보
- 상품명 (최대 2줄, 말줄임)
- 쿠폰할인 원가 (취소선, 조건부)
- 할인율 뱃지 + 판매가 Bold
- 단위당 가격 (예: "10g당 151원")
- 별점 + 리뷰 수
- 클릭 시 상품 상세로 이동

### 4.5 무한 스크롤
- 스크롤 하단 도달 시 다음 페이지 자동 로딩
- 로딩 중 Skeleton UI or 스피너
- 마지막 상품 시 "마지막 상품입니다" 안내

### 4.6 플로팅 버튼 → 구현 X

## 5. 상품 상세 `/products/:productId`

### 5.1 헤더
- 좌측 뒤로가기 버튼

### 5.2 상품 이미지 슬라이더
- Full Width 노출
- 스와이프 이미지 전환
- 하단 인디케이터 도트
- 우측 하단 하트(♡) 찜하기 토글
  - 비로그인: 로그인 페이지 리다이렉트
  - 로그인: 찜 추가/해제 API

### 5.3 상품 기본 정보
- 구매 통계 + 우측 공유 아이콘 (네이티브 Share API)
- 브랜드 로고 + 브랜드명 (클릭 시 브랜드 상품 리스트)
- 상품명 (최대 2줄, 말줄임)
- 할인율 뱃지 + 판매가 Bold + 원가 취소선

### 5.4 옵션 선택
- 이미지형 옵션 카드 (색상 x 수량 조합)
- 선택된 옵션: 파란색 테두리 하이라이트
- "인기" 뱃지: 가장 많이 선택된 옵션
- "모든 옵션 보기 >" → 전체 옵션 바텀시트
- 라디오 버튼 방식 수량 단위 선택

### 5.5 상품 한눈에 보기 → 구현 X (AI 요약 X)

### 5.6 상품 정보 이미지
- 판매자 등록 이미지 순서대로 노출
- Lazy Loading 처리

### 5.7 관련 상품 섹션 → 구현 X

### 5.8 상품 리뷰
- 전체 평점 + 리뷰 수
- "리뷰 작성하기": 로그인 + 구매 이력 있는 경우만 활성화
- AI 리뷰 요약 → 구현 X
- 자주 언급된 키워드 → 구현 X
- 리뷰 미디어 갤러리 (가로 스크롤)
- 리뷰 카드 리스트 (별점/닉네임 마스킹/날짜/본문/"더보기")
- "도움이 됐요 👍" 버튼 → API 호출

### 5.9 상품 문의/배송·교환·반품 → 구현 X

### 5.10 하단 고정 CTA
- 장바구니 담기 버튼 (Outlined): POST /api/cart
- 바로구매 버튼 (Primary, Blue): 옵션 바텀시트 → 주문서 페이지
- 옵션 미선택 시 바텀시트 자동 노출

### 5.11 로그인 상태 분기
- 비로그인: 찜/장바구니/구매/리뷰 작성 클릭 시 로그인 리다이렉트

## 6. 검색 `/search`

### 6.1 헤더
- 좌측 뒤로가기
- 검색 입력 필드 (진입 시 Auto Focus)
- X 버튼: 입력값 초기화
- Enter → 검색 결과 페이지 이동
- 우측 카메라 아이콘 → 이미지 검색

### 6.2 최근 검색어
- 로그인: 서버 기반 동기화
- 비로그인: localStorage 기반
- 칩 형태 가로 스크롤
- 전체 삭제 / 자동 저장 끄기

### 6.3 추천 검색어
- API 관리 키워드, 칩 형태
- Wrap 방식 다중 행 (가로 스크롤 X)
- "i" 버튼 → 추천 기준 안내 Tooltip or 모달

### 6.4 검색어 입력 중 자동완성
- 1자 이상 입력 시 드롭다운 노출
- 입력 키워드 Bold 강조
- Debouncing 300ms

## 7. 챗봇 `/chatbot`

### 7.1 헤더
- 좌측 뒤로가기
- 타이틀: "쇼핑 도우미"

### 7.2 대화 영역
- 웰컴 메시지 + Quick Reply 버튼
- 말풍선: 봇(좌측, 회색) / 사용자(우측, 파란색)
- 메시지 하단 발송 시각
- URL → 클릭 가능한 링크 자동 변환
- 로딩 인디케이터: 점 세 개 애니메이션 (...)
- 새 메시지 수신 시 자동 최하단 스크롤
- "새 메시지 ▼" 버튼

### 7.3 상품 카드 응답
- 상품 썸네일 / 상품명 / 할인율+판매가 / 배송 뱃지 / 별점+리뷰 수
- "상품 보기" → 상품 상세
- "장바구니 담기" → API 호출
- 다수 추천 시 가로 스크롤 카드 리스트

### 7.4 빠른 답변 버튼 (Quick Reply)
기본 진입 시 예시:
- 상품 추천해줘 / 주문 조회 / 배송 문의 / 반품·교환 문의 / 자주 묻는 질문

### 7.5 메시지 입력 영역
- 하단 고정
- Placeholder: "메시지를 입력하세요..."
- 최대 500자
- 높이 자동 조절 (최대 4줄)
- 전송 버튼: 입력값 없을 시 비활성화

## 8. 장바구니 `/cart`

### 8.1 헤더
- 좌측 뒤로가기, 타이틀 "장바구니"

### 8.2 탭 영역 → 구현 X

### 8.3 전체선택/선택삭제
- 전체선택 체크박스 (전체 선택/해제 토글)
- 선택삭제 버튼 (1개 이상 선택 시 활성화) → 확인 모달 후 일괄 삭제 API

### 8.4 배송 그룹 섹션 → 구현 X

### 8.5 상품 카드
- 좌측 체크박스 (개별 선택/해제)
- X 버튼 → 삭제 확인 모달 → 단일 삭제 API
- 상품명 (최대 1줄, 말줄임)
- 선택된 옵션 텍스트
- 쿠폰할인 원가 취소선 (조건부)
- 할인율 뱃지 + 판매가 Bold
- 수량 조절 스테퍼 (— / +)
  - — : 수량 1일 때 비활성화
  - + : 최대 수량 초과 시 비활성화 + Toast
  - 수량 변경 시 API 호출 + 가격 합계 실시간 갱신

### 8.6 할인쿠폰 → 구현 X
### 8.7 와우회원 혜택 배너 → 구현 X

### 8.8 결제 금액 요약
- 총 상품 가격 / 총 쿠폰할인 (빨간색) / 총 배송비 / 총 결제 예상 금액 (Bold)
- 선택 상품 변경 시 실시간 갱신

### 8.9 하단 고정 CTA
- 원가 취소선 + 최종 결제 금액 Bold
- ∧ 버튼: 결제 금액 상세 펼치기/접기
- "총 N개 상품 구매하기" 버튼 (선택 0개 시 비활성화)
- 클릭 시 주문서 페이지로 이동

### 8.10 빈 바구니 상태
- 일러스트 + "장바구니가 비어 있어요"
- "쇼핑 계속하기" → 메인 페이지

## 9. 마이페이지 `/mypage`

### 9.1 유저 프로필
- 프로필 이미지 (기본 아바타, 미설정 시 이니셜)
- 닉네임/이름 마스킹 (예: "임*현")
- 설정(⚙️) 아이콘 → 계정 설정 페이지
- 비로그인: "로그인 / 회원가입" 버튼 노출

### 9.2 멤버십/혜택 카드 → 구현 X

### 9.3 퀵 메뉴 아이콘 (5개)
| 메뉴 | 이동 페이지 |
|------|------------|
| 주문내역 | 전체 주문 내역 리스트 |
| 찜리스트 | 위시리스트 페이지 |
| 최근본상품 | 최근 본 상품 리스트 |
| 자주산상품 | 반복 구매 상품 리스트 |
| 전체메뉴 | 전체 메뉴 페이지 |

### 9.4 주문 내역 섹션
- 섹션 타이틀 + "전체 보기 >" → 주문 내역 전체 페이지
- 최근 주문 카드 가로 스크롤
- 배송 상태 색상: 배송완료(검정) / 배송중(파란색) / 주문접수(주황색)
- 장바구니 담기 버튼 (POST /api/cart)
- API: GET /api/mypage/orders?limit=5

### 9.5 프로모션 배너 → 구현 X
### 9.6 최근 찾던 상품의 연관 상품 → 구현 X
